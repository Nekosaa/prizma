# MODERNIZATION_GUIDE.md

Подробное описание того, что и зачем было сделано в рамках модернизации Prizma Studio.

## 1. Цели

* Уйти от «Notebook на 4 вкладки» к раскладке в стиле IDE / браузера.
* Дать возможность открывать **несколько файлов одновременно**.
* Показать пользователю осмысленный **стартовый экран**.
* Улучшить визуальное восприятие журнала событий.
* Сохранить **100 % обратной совместимости** с оригинальным `main.py`.

## 2. Ключевые улучшения

### Sidebar (240 px)
* `core/modern_widgets.py::ModernSidebar`
* Хранит список недавних файлов в `~/.tools_config.json` (до 8 записей)
* Клик по элементу открывает файл в новой вкладке
* Наведение подсвечивает акцентным цветом

### Browser-style Tabs
* `core/modern_widgets.py::BrowserTabBar`
* Каждый открытый файл — отдельная вкладка
* Кнопка `×` закрывает конкретную вкладку
* Иконки: `📄` PDF, `🎨` PSD, `⚙` Настройки, `ℹ` О программе

### Welcome Screen
* `core/modern_widgets.py::WelcomeScreen`
* Показывается, когда нет открытых файлов
* Две акцентные кнопки для быстрого старта

### Top Bar
* Слева — акцентные кнопки открытия
* Справа — Ghost-кнопки Settings/About + выбор языка

### Modern Log Panel
* `core/modern_widgets.py::ModernLogPanel`
* Каждый уровень (info/ok/warn/error) — свой символ и цвет
* Цвета берутся из активной палитры

### Drag & Drop (опционально)
* Требует `tkinterdnd2`
* Если пакет не установлен — интерфейс работает без DnD

### Современная палитра
* `core/theme_modern.py`
* Спокойный синий акцент (`#2E7DFF`) вместо «AI-фиолетового»
* Отдельные цвета для success / warning / danger

## 3. Что осталось без изменений

* `main.py` — оригинальный entry point
* `core/theme.py` — используется классической версией
* `modules/pdf_tools_tab.py` и `modules/psd_tools_tab.py`
* `assets/`, `build_exe.py`, `PrizmaStudio.spec`, `fix_unicode.py`

## 4. Технические детали

* Каждая вкладка — **новый экземпляр** `PdfToolsFrame`/`PsdToolsFrame`.
* Активная вкладка кладётся в `panel_host` через `.grid()`,
  остальные скрываются `grid_forget()` (не пересоздаются).
* Служебные экраны Settings/About тоже открываются как вкладки.
* Смена языка/темы применяется мгновенно через `i18n.subscribe(...)`.
